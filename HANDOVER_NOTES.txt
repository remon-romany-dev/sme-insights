================================
3. Theme Capabilities (SME Insights Theme)
================================

This is a production-ready publishing platform with enterprise-level features:

- **Custom Content Architecture**: Custom post types, taxonomies, and flexible content system without requiring ACF or premium plugins.
- **SEO & Performance**: Built-in Schema.org markup, XML sitemap generation, lazy loading, critical CSS, and optimized database queries.
- **Frontend Quick Editor**: Edit text and content directly from the frontend with hover buttons for rapid updates.
- **Image Optimization**: Automatic compression and optimization on upload without external dependencies.
- **Multiple Editing Workflows**: Supports Quick Edit, Visual Editor, and Gutenberg blocks to match any team's workflow.
- **Coming Soon Mode**: Built-in maintenance page with countdown timer and email subscription.

================================
4. Plugin Capabilities (SME Insights Generator)
================================

This is not just a simple connector; it is a robust content engine featuring:

- **Multi-Model Support**: Works with OpenAI (GPT-4o/Turbo), Google Gemini (1.5 Pro/Flash), and Anthropic Claude (3.5 Sonnet).
- **Smart Fallback System**: Automatically switches to Gemini if OpenAI quota is exceeded to ensure uninterrupted operation.
- **Automated Batch Processing**: Runs in the background via WP Cron to generate multiple articles at set intervals.
- **Dynamic Image Handling**: Automatically selects relevant images from Unsplash and assigns proper alt text.
- **SEO Optimization**: Auto-generates meta descriptions, excerpts, and Open Graph tags for every article.

================================
4. Quick Installation Guide
================================

1. **Theme Installation:**
   - Go to Appearance > Themes > Add New > Upload Theme.
   - Upload `sme-insights-theme.zip` and activate it.

2. **Plugin Installation:**
   - Go to Plugins > Add New > Upload Plugin.
   - Upload `sme-insights-generator.zip` and activate it.
   - Go to the new "Content Generator" menu item to configure API keys.
